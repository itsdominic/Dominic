About Me

I go by the name of Dominic Charles but you probably already guessed that by the huge showcase. I’m a recent graduate from the University of Hertfordshire currently employed as an IT Coordinator. Whilst on the subject university during my studies I took both a Web Scripting & Advanced Web Scripting module which gave me exposure to the world of Web Development. From these modules, I learned the basics of HTML, CSS, and the fundamentals JavaScript and I quickly discovered Web Dev was my passion.

After leaving university I continued to advance my knowledge independently. I added the use Git and GitHub, Bootstrap 4, and the CSS predecessor SASS to my workflow which I learnt all in my own time. I also developed my core CSS knowledge by learning CSS grid and Flexbox which makes creating page layouts a breeze. Me and JavaScript have a love-hate relationship however learning how to manipulate the DOM to do some cool things in the browser was another integral part of my development. Looking further afield I want to hone in on my skills in an official Web Development role and eventually learn a front-end framework such as React as well as some backend technologies such as Node.js and Express.
